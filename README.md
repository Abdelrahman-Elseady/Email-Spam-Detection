# Spam Email Detection

This project is a machine learning-based system for detecting spam emails. It processes raw email text data through tokenization and feature extraction using TF-IDF (Term Frequencyâ€“Inverse Document Frequency). The goal is to classify emails as **spam** or **ham** (not spam) effectively.

## Features

-  **Text Preprocessing**
  - Lowercasing
  - Removal of punctuation and special characters
  - Stop words removal
  - Tokenization

- **Feature Extraction**
  - TF-IDF vectorization for converting text into numerical features
